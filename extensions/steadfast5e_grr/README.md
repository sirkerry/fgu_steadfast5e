# Steadfast5e - Gritty Realism Rest

![Steadfast5e - Gritty Realism Rest](forge/steadfast5e_grr.png)

**FG Forge Listing:** not yet published

Part of the [Steadfast5e](https://forge.fantasygrounds.com/) suite for grittier, OSR-style 5E play.

Overhauls 5E rest and recovery to create meaningful attrition: configurable short/long rest durations, limited HD spending, optional CON/Endurance check for HD recovery after long rest, fractional HP recovery, and scope controls per creature type. Optional integration with Steadfast5e - Location System for per-tier overrides.
