# Steadfast5e — Expanded Skills (Reference Module)

Companion reference module for the Steadfast5e Expanded Skills extension. Provides library entries for all 27 skills in the Steadfast5e skill system, visible to both GM and players once activated.

## The 27 Skills

**Strength:** Athletics, Coercion
**Dexterity:** Acrobatics, Stealth, Trickery
**Intelligence:** Academics, Arcana, Crafting, Investigation, Nature, Religion, Science
**Wisdom:** Beastcraft, Cooking, Healing, Herblore, Insight, Occult, Survival, Tracking, Wildcraft
**Constitution:** Endurance, Perception
**Charisma:** Guile, Perform, Persuasion, Streetwise

Each entry includes a description of what the skill covers and why it uses its ability score, explicit scope boundaries against adjacent skills, five example uses, and GM guidance for when to call for the skill.

## How to Use

1. Place `steadfast5e_expanded_skills.mod` in your Fantasy Grounds Unity `modules/` directory.
2. Open a campaign with the Steadfast5e Expanded Skills extension loaded.
3. Go to Library → Modules and activate **Steadfast5e Expanded Skills**.
4. The 27 skill entries appear under the Skills section of the library, ready to reference during play.

FGU saves the activation state per campaign — this only needs to be done once.

## Requires

The **Steadfast5e Expanded Skills** extension — this module is reference-only and does not affect any mechanics on its own.
